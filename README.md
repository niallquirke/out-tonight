# out-tonight

## Usage

### Out Tonight Endpoint

Who's out tonight and who isn't

```
https://api.whosouttonight.ie/out-tonight
```

**To see who's out tonight**

```
https://api.whosouttonight.ie/out-tonight
```

**To add a new person**

```
https://api.whosouttonight.ie/out-tonight?name=donal&out=true
```
